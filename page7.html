PROPUESTA FINAL

Sistema de monitoreo continuo inteligente para diagnóstico de taquicardias ventriculares. Este dispositivo cuenta con una batería de litio de 1000mAh recargable con cable micro USB que le brindará la autonomía para monitoreo por periodos largos de tiempo. Además, para la adquisición de las señales se opta por los electrodos capacitivos, también llamados textiles, que no tienen contacto directo con la piel ya que está recubierto con una tela de algodón para mayor comodidad. Para el procesamiento se desarrollará un sistema de diagnóstico con machine learning basado en criterios de detección ya existentes y se testeará en la base de datos del MIT-BIH, la cual contiene tanto ECG de holters como dispositivos de monitoreo ambulatorio que nos permitirá entrenar al programa.
