Plotting ECG data

Pan Tompkings Algorithm

  1. Main.py - Este es el programa principal del proyecto en el cual se utilizan las librerías creadas en el directorio para subir, crear, descargar, modificar, eliminar y buscar archivos tanto en la targeta SD y al nube de Google Drive. Esto nos permitirá crear toda la base de datos para almacenar la información y permitir su visualización por el doctor. Además, en este programa se leerán los datos adquiridos por el ECG y se empleará la librería que se creó para la detección de picos para diagnósticar la Taquicardia Ventricular. Finalmente, todos estos datos se almacenarán dependiendo del diagnóstico y si se ha presionado el botón de pánico, esto también estará intercomunicado con una Android App  . Try it now
  2. Pan_Tompkings_algorithm.py - Este algoritmo detecta los picos S maximizando la presencia del complejo QRS aplicando un filtro pasabanda, luego se aplica una derivada para obtener información de la pendiente, se eleva al cuadrado para obtener valores positivos e incrementarlos, finalmente mediante un integrador de ventana móvil se haya el pico S mediante el cual usándolo de referencia se pueden hallar los demás picos utilizando transformadas de fourier . Try it now
  3. Storing.py - Mediante códigos open source de Google Developers y la posibilidad de modificar OS con comandos, se desarrolla un programa con la integración de Google Drive API para la modificación de archivos en una plataforma de Google Drive y el propio directorio en la SD. Try it now
